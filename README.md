# Habit Tracker

This is a simple Habit Tracker web application that helps users build and maintain daily habits.

The project is built using basic web technologies like HTML, CSS, and JavaScript. It allows users to add habits, mark them as completed, and track their overall progress in a clean and simple interface.

## Features

- Add new daily habits
- Mark habits as completed
- Delete habits
- Progress bar to track completion
- Reset all habits
- Dark mode option
- Data saved using browser localStorage

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

1. Type a habit in the input field.
2. Click the **Add** button.
3. Check the box when the habit is completed.
4. The progress bar will automatically update.
5. Habits are saved in the browser using localStorage.
